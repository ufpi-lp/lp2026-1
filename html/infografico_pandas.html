pandas

A biblioteca essencial para análise, limpeza, exploração e manipulação de dados no Python 🐍

O Pandas é uma biblioteca open-source usada para trabalhar com conjuntos de dados em memória. Ele combina estruturas de dados eficientes, leitura de múltiplos formatos, limpeza, transformação, sumarização estatística e integração com NumPy, Matplotlib e ferramentas de Ciência de Dados.

SeriesDataFrameCSV / Excel / SQL / JSONLimpeza de dadosEDAVisualizaçãoMachine Learning

📊

Transforma dados brutos em tabelas prontas para análise.

🧹

Facilita limpeza, correção, padronização e remoção de duplicidades.

⚙️

Permite filtros, seleção, agregações, ordenação e transformação.

📈

🤖

Leitura e escrita de dados

O Pandas facilita a importação de dados de várias fontes e também a exportação dos resultados processados.

📄
CSV

Arquivos separados por vírgula, ponto e vírgula ou outro delimitador.

pd.read_csv()
📗
Excel

Planilhas .xlsx e .xls com abas e tabelas.

pd.read_excel()
🗄️
SQL

Consulta a bancos relacionais por conexão.

pd.read_sql()
🧾
JSON

Dados semiestruturados de APIs e arquivos.

pd.read_json()
🌐
HTML

Tabelas extraídas de páginas web.

pd.read_html()

Exemplo: lendo um CSV

import pandas as pd

df = pd.read_csv(
    'dados.csv',
    sep=';',
    encoding='utf-8'
)

print(df.head())

Limpeza e preparação dos dados

Dados ausentes

Identifique células vazias ou valores NaN e escolha uma estratégia.

df.isnull().sum()
df.dropna()
df.fillna(0)

Duplicidades

Remova registros repetidos que podem distorcer os resultados.

df.duplicated().sum()
df = df.drop_duplicates()

Tipos de dados

Converta colunas para tipos adequados: número, texto ou data.

df['data'] = pd.to_datetime(df['data'])
df['votos'] = df['votos'].astype(int)

Padronização textual

Padronize maiúsculas, espaços e acentos conforme a necessidade.

df['municipio'] = df['municipio'].str.strip().str.upper()

Renomear colunas

Use nomes curtos e claros para simplificar consultas e gráficos.

df = df.rename(columns={
    'NM_MUNICIPIO': 'municipio',
    'QT_VOTOS': 'votos'
})

Salvar resultado limpo

Exporte o DataFrame tratado para reuso em análises posteriores.

df.to_csv('dados_limpos.csv', index=False)
df.to_excel('dados_limpos.xlsx', index=False)

Manipulação e transformação de dados

Selecionar colunas

Escolha uma ou mais colunas relevantes para análise.

df['votos']
df[['municipio', 'votos']]

Filtrar linhas

Use condições booleanas ou consultas com query.

df[df['votos'] > 1000]
df.query("municipio == 'TERESINA'")

Ordenar dados

Organize registros por uma ou mais colunas.

df.sort_values('votos', ascending=False)

Criar colunas

Gere variáveis derivadas a partir de outras colunas.

df['percentual'] = df['votos'] / df['votos'].sum() * 100

Agrupar dados

Resuma dados por categoria, município, partido ou período.

df.groupby('municipio')['votos'].sum()

Excluir colunas/linhas

Remova atributos ou registros que não serão usados.

df.drop(columns=['coluna_inutil'])
df.drop(index=[0, 1])
6

Operações avançadas

Merge / Join

Combina DataFrames usando chaves, como em bancos SQL.

pd.merge(df_votos, df_cidades, on='municipio')

Concatenação

Empilha DataFrames verticalmente ou horizontalmente.

pd.concat([df_jan, df_fev], axis=0)

Pivot Table

Cria tabelas dinâmicas para resumir dados por categorias.

pd.pivot_table(df, values='votos', index='municipio', columns='cargo', aggfunc='sum')

Apply

Aplica funções personalizadas em colunas ou linhas.

df['faixa'] = df['idade'].apply(lambda x: 'adulto' if x >= 18 else 'jovem')

Datas

Extrai ano, mês, dia e calcula intervalos temporais.

df['ano'] = df['data'].dt.year
df['mes'] = df['data'].dt.month

Janelas móveis

Calcula médias, somas e tendências em séries temporais.

df['media_7d'] = df['vendas'].rolling(7).mean()

Correlação

Identifica relações lineares entre variáveis numéricas.

df.corr(numeric_only=True)

Pipeline

Encadeia transformações para criar fluxos reprodutíveis.

df_limpo = (df
  .dropna()
  .rename(columns=str.lower)
  .sort_values('votos'))

Novos exemplos práticos relacionados

Exemplo A — análise de votos por candidato

Inspirado em dados eleitorais: selecionar colunas, filtrar município, agrupar por candidato e calcular total de votos.

Objetivo: descobrir os candidatos mais votados em uma cidade e criar uma tabela ordenada.
colunas = ['NM_MUNICIPIO', 'NM_VOTAVEL', 'QT_VOTOS']
teresina = df[colunas].query("NM_MUNICIPIO == 'TERESINA'")

ranking = (teresina
    .groupby('NM_VOTAVEL', as_index=False)['QT_VOTOS']
    .sum()
    .sort_values('QT_VOTOS', ascending=False))

print(ranking.head(10))

Exemplo B — análise mensal de temperatura

Criação de DataFrame a partir de dicionário, cálculo estatístico e gráfico simples.

import pandas as pd
import matplotlib.pyplot as plt

meses = ['Jan','Fev','Mar','Abr','Mai','Jun']
temperaturas = [31, 30, 29, 27, 25, 24]

df_temp = pd.DataFrame({
    'mes': meses,
    'temperatura': temperaturas
})

print(df_temp.describe())
df_temp.plot(x='mes', y='temperatura', kind='line', marker='o')
plt.title('Temperaturas no semestre')
plt.show()

Boas práticas para trabalhar com Pandas

🔍

Conheça seus dados

Use head(), info(), describe() e value_counts() antes de transformar.

🧼

Limpe antes de analisar

Dados ausentes, tipos errados e duplicidades podem comprometer conclusões.

🏷️

Nomeie bem

Prefira colunas em minúsculas, sem espaços e com nomes claros.

📌

Use .loc

Evite ambiguidades ao alterar subconjuntos do DataFrame.

🧾

Documente passos

Comentários e pipelines tornam a análise mais reprodutível e didática.

Resumo

Pandas é uma ferramenta central para quem trabalha com dados em Python. Ele permite carregar, explorar, limpar, transformar, resumir e visualizar dados de forma eficiente e produtiva.

Quando usar?

  • Análise exploratória de dados.
  • Limpeza e preparação de datasets.
  • Relatórios, gráficos e tabelas resumidas.
  • Pré-processamento para Machine Learning.

Mini checklist

  • Carreguei os dados corretamente?
  • Verifiquei tipos, nulos e duplicados?
  • Filtrei somente as colunas necessárias?
  • Salvei o dataset limpo?
