import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 10, 400)
y = np.sin(x)

plt.figure(figsize=(7, 4))
plt.plot(x, y, color='royalblue', linewidth=2, label='sin(x)')
plt.title('Função Seno')
plt.xlabel('x (rad)')
plt.ylabel('y')
plt.grid(True, linestyle='--', alpha=0.6)
plt.legend()
plt.tight_layout()
plt.show()
Função Seno

Principais tipos de gráficos

Linha

Mostra evolução de valores contínuos.

plt.plot(x, y)

Dispersão

Exibe relação entre duas variáveis.

plt.scatter(x, y)

Barras

Compara categorias com valores.

plt.bar(categorias, valores)

Barras horizontais

Útil para rótulos longos.

plt.barh(y, x)

Pizza

Mostra proporções de um todo.

plt.pie(valores)

Histograma

Mostra distribuição de frequências.

plt.hist(dados)

Boxplot

Resume mediana, quartis e outliers.

plt.boxplot(dados)

3D

Visualizações tridimensionais.

projection='3d'

Personalização do gráfico

  • Título: plt.title('Título')
  • Rótulos: plt.xlabel('X') e plt.ylabel('Y')
  • Limites: plt.xlim() e plt.ylim()
  • Grade: plt.grid(True)
  • Cores e estilos: color, linestyle, marker
  • Legenda: plt.legend()
  • Anotações: plt.annotate()
Exemplo personalizado

Subplots: múltiplos gráficos

fig, axs = plt.subplots(2, 2, figsize=(8, 6))

axs[0,0].plot(x, np.sin(x))
axs[0,0].set_title('Seno')

axs[0,1].plot(x, np.cos(x))
axs[0,1].set_title('Cosseno')

axs[1,0].plot(x, x**2)
axs[1,0].set_title('x²')

axs[1,1].plot(x, np.log1p(x))
axs[1,1].set_title('log(1+x)')

plt.tight_layout()
plt.show()

Integração com Pandas

import pandas as pd

# Criar um DataFrame
dados = {
  'Mês': ['Jan', 'Fev', 'Mar', 'Abr', 'Mai', 'Jun'],
  'Vendas': [120, 150, 190, 170, 210, 230]
}
df = pd.DataFrame(dados)

# Gráfico de linha
df.set_index('Mês')['Vendas'].plot(marker='o')
plt.title('Vendas Mensais')
plt.ylabel('Vendas (R$)')
plt.grid(True, linestyle='--', alpha=0.6)
plt.show()
Vendas Mensais

Boas práticas

  • ✓Escolha o tipo de gráfico adequado ao objetivo da análise.
  • ✓Use títulos claros e rótulos descritivos nos eixos.
  • ✓Mantenha design limpo e consistente.
  • ✓Use legendas somente quando forem necessárias.
  • ✓Evite excesso de elementos visuais no mesmo gráfico.
  • ✓Prefira cores com contraste e paletas acessíveis.
  • ✓Salve gráficos em alta resolução para publicações.
  • ✓Documente o código para facilitar manutenção e replicação.

Dicas importantes

  • plt.show() exibe o gráfico na tela.
  • plt.close() fecha a figura atual e libera memória.
  • figsize=(largura, altura) controla o tamanho da figura.
  • plt.tight_layout() ajusta espaçamentos automaticamente.
  • dpi define a resolução ao salvar imagens.
